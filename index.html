Rangkuman Materi GIT

  1. Membuat "Git init"
  2. Kemudian "Git Add ." file yang akan di commit
  3. Setelah itu "Git Commit -m "file-commit" "
  4. Sesudah Commit, pilih branch "Git branch -M main"
  5. lalu Sambungkan ke repo di gitHub "git remote add origin https://github.com/yuliantocode93/materi-git.git"
  6. setelah tersambung ke repo gitHub lalu push file "Git push -u origin main"
  7. Dan refresh halaman repository gitHub
  8. Untuk mengecek status commit file "Git Status atau Git log
